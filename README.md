PricePulse app for tracking spot electricity prices.

## Technologies Used
- JavaScript
- React.js
- Vite
- API integration

## Installation & Setup

Follow these steps to run the application:

1. Clone the repository to your local machine.
2. Open a terminal in the project directory.
3. Run the following command in the terminal to start the application:
    1. npm install
    2. npm run dev

## Main components

**App.jsx:** The main application component that sets up the routes and navigation.
**HomePage.jsx:** Displays the current electricity price and provides navigation to other pages.
**PriceChartPage.jsx:** Displays a chart of electricity prices over different time periods.
**AppliancesPage.jsx:** Lists the appliances and their electricity consumption.
**AddAppliance.jsx:** Form to add a new appliance.
**Chart.jsx:** Component to render the price chart using react-chartjs-2.
**PriceTable.jsx:** Component to render a table of recent price changes.
**Navbar.jsx:** Bottom navigation bar for the application.

## Configuration
**vite.config.js:** Configuration file for Vite.
**eslint.config.js:** Configuration file for ESLint
